## 开源项目表达原则

页面中的开源项目均以“业务落地方案”表达，不声称作者身份：

- `已落地`：已有真实仓库、流程、文档或可验证产物。
- `原型中`：已完成字段、流程或方案设计，但未完整上线。
- `参考方案`：仅作为产品理解或技术路线参考。

除 `ops-codex-workflow-library` 外，不把第三方开源项目包装成个人原创项目。
