# 运营周报飞书汇报措辞

把目标写成数字，把复盘写成结论，把规划写成排期，输出能直接粘贴进飞书的中文版本。

## Trigger

Use this skill when the user asks for: 周报整理、汇报措辞.

## Inputs

- 本周工作记录
- 下周规划
- 已有飞书表格结构

## Workflow

- 先把目标改成可衡量数字。
- 再把复盘压缩成结论，不写流水账。
- 最后把下周事项补成周二到周一的排期表。

## Failure Handling

- 业务数字缺失时标出缺口，不虚构结果。

## Human Review Boundary

不替团队承诺未经确认的目标和日期。

## Expected Outputs

- 可粘贴周报
- 周二到周一排期表
