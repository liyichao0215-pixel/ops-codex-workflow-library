# Flova 教程视频协同包

这是一套把教程视频制作中的高频返工问题整理成 Skill，并把人与 Codex 的协作过程整理成 SOP 的可分享资料。

## 内容

- `SOP.md`：给人看的完整执行流程，包含素材提交、确认、反馈、返修和交付。
- `给朋友的领取提示词.md`：发给朋友的使用说明和 Codex 启动提示词。
- `skill/flova-tutorial-video-coordination/SKILL.md`：给 Codex 使用的可复用 Skill。
- `skill/flova-tutorial-video-coordination/references/问题诊断表.md`：常见问题的定位、修复和验收表。

## 适用场景

FlovaAI、AI 工具、软件功能、产品页面和剪辑软件教程视频，尤其适合同时包含口播音频、字幕、录屏、截图、生成视频和动效包装的项目。

## 最重要的一句话

声音定节奏，字幕定理解，素材定可信度，动效定爽感；每次返修只改有时间码的问题，局部通过后再渲染全片。

## 给朋友怎么用

让朋友先读 `给朋友的领取提示词.md`，再把 `skill/flova-tutorial-video-coordination` 放进自己的 Codex Skills 目录。第一次使用时，把里面的提示词直接发给 Codex 即可。
