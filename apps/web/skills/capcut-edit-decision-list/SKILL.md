# 教程脚本转剪映执行清单

把教程脚本拆成时间线、屏幕动作、字幕、强调点和导出检查项，交给剪辑阶段使用。

## Trigger

Use this skill when the user asks for: 教程剪辑规划、字幕和重点标注.

## Inputs

- 教程脚本
- 录屏素材列表
- 目标平台

## Workflow

- 按章节拆分脚本和屏幕动作。
- 标记缩放点、重点框选、字幕断句和音效时机。
- 输出剪映可照着执行的编辑决策清单。

## Failure Handling

- 没有素材文件名时先用占位名，剪辑前再替换。

## Human Review Boundary

不替代最终人工观感检查。

## Expected Outputs

- 剪辑决策表
- 字幕检查清单
