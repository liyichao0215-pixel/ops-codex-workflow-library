# 活动/内容运营选题复盘转下周排期

## 使用场景

当运营同事有本周数据、用户反馈、问题记录或活动复盘，但还没有形成下周排期时使用。

## 工作步骤

1. 把本周内容分成有效动作、无效动作和不确定动作。
2. 提炼能复用到下周的选题方向或活动动作。
3. 生成负责人、时间、输入材料和验收口径。
4. 标出缺少的业务数字和需要人工确认的事项。

## 输出格式

```text
本周有效结论：
本周无效动作：
下周选题/动作：
负责人：
排期：
待补数据：
验收口径：
人工确认项：
```

## 边界

不要虚构业务数据，不替团队承诺未经确认的日期、负责人、预算或业务结果。
